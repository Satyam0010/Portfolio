# Build a Premium Software Engineer Portfolio

Design and develop a **world-class personal portfolio website** for me.

The goal is to impress recruiters and software engineers at companies like Google, Microsoft, Amazon, Atlassian, Adobe, Stripe, Vercel, Cloudflare, and other top product-based companies.

This website should **not look like a typical developer portfolio** or an AI-generated template.

Instead, it should feel like a premium SaaS landing page that reflects engineering excellence, attention to detail, and modern design principles.

Think of the design language of:

* Apple
* Linear
* Vercel
* Stripe
* Raycast
* Notion

The first impression should communicate:

> Professional • Minimal • Modern • Elegant • Memorable

---

# Design Philosophy

Less is more.

Prioritize typography over decoration.

Use whitespace generously.

Every section should have purpose.

Avoid visual clutter.

Do not use generic portfolio layouts.

Do not use stock illustrations.

Do not use large glowing gradients.

Do not use random floating blobs.

Do not use excessive glassmorphism.

Do not use generic "developer" graphics.

Avoid anything that immediately makes visitors think "AI-generated portfolio."

Everything should feel handcrafted.
